## 📌 Sobre o projeto

Este projeto foi desenvolvido para auxiliar o registro e acompanhamento de pessoas desaparecidas durante situações de enchentes. Esse sistema permite cadastrar indivíduos desaparecidos, armazenar suas informações e atualizar seu status conforme novas informações, facilitando a organização.
---

---

Backend

- SQLite

Frontend

- React
